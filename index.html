How does javascript work?

JavaScript is a single-thread synchronous language. But sometimes acts asynchronously. The browser has an engine running Chrome V8 to run JavaScript, using the SpiderMonkey on Mozilla, using the Safari Webkit. The source code is gone through a program called a compiler, which makes an interpretation of it into byte codes that the machine can comprehend and work on. On the other hand, JavaScript has no compiler steps. All things being equal, a translator in the program peruses the JavaScript code, parses each line, and executes it.

Difference between local storage and session storage?
Local storage
  • Local storage is not session-based, this data didn't expire.
  • Local storage takes up a maximum of 10 MB.
  • Accessible through document.localStorage
Session storage
  • Session storage is session-based, if you close your browser tab then Session storage will be deleted.
  • Session storage takes up a maximum of 5 MB.
  • Accessible through document.sessionStorage
